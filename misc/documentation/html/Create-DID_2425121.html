ASTPP Documentation : Create DID

Created by ASTPP, last modified on Jul 29, 2015

You can create new DID using below page,

DIDs Add/Edit Field description

DID Information

DID

Enter unique numeric DID number

Country

Select country of DID

City

DID City

Province/State

DID State

Provider

Select provider to whom this DID belongs to

DID Setting

Call Type

LOCAL : Wish to route call to Local extension

 PSTN : Wish to route call to PSTN Number

OTHER : If you wish to route call to custom destination

Destination

Set appropriate destination based on call Type.

Example :

LOCAL : 1001 (Local Extension number)

PSTN : 1800214018 (PSTN number)

OTHER : sofia/default/1234567890@192.168.1.3

OR

sofia/gateway/gwname/121423232

Max ChannelsMaximum allowed concurrent channels for DID calls. 0=Unlimited


DID Billing

AccountSelect account number you wish to assign DID
Increments

Rate of increment to calculate call cost. 

Example : 60 to charge every minute

CostCost per minute
Included SecondsDefine seconds will be free from the call duration for each call
Setup FeeOne time Setup fee
Monthly FeeMonthly recurring fee
Connection FeeConnection fee to charge customer minimum when their call will be connected
