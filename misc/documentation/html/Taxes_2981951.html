ASTPP Documentation : Taxes

Created by ASTPP, last modified on Jul 21, 2015

Taxes are meant to be charge some amount on invoices. After creating taxes you can assign it to any of customer by clicking on Add Taxes button from customer list.

Taxes can assign to customer as well as resellers. You can specify tax in percentage (%) as well as specific amount to be charge on invoice.

To create Taxes go to Accounting menu in ASTPP portal. From popup select Taxes it will show you the list of the taxes. You can create new taxes by clicking on Create button.
