ASTPP Documentation : Create Termination Rate

Created by ASTPP, last modified on Jul 30, 2015

TrunkSelect trunk for termination rate
CodePrefix of termination rate. Example: 91
DestinationDescription for rate. Example : India
StripTo remove any specific prefix from dialed number
PrependTo append any specific prefix in dialed number
Connect CostConnection fee to charge customer minimum when their call will be connected
Included SecondsDefine seconds will be free from the call duration for each call
Per Minute CostCost per minute
Increment

Rate of increment to calculate call cost

Example : 60 to charge every minute

Precedence

Priority of rate. If LCR found same prefix with same rate for different trunk then based on precedence/priority level trunk will be selected.
