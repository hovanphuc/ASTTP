ASTPP Documentation : Import Origination Rate

Created by ASTPP, last modified on Jul 29, 2015

File must be in the following format(.csv):

Code,Destination,Connect Cost,Included Seconds,Per Minute Cost,Increment,Precedence.

91,India,0,0,0.013,60,0

1,USA.,0,0,0.005,60,0

Select rate group, force trunk and csv file of origination rates and click on Import button, It will show below output to confirm if rates format are correct. Once you confirm then click on process and it will import those rates in system.
