ASTPP Documentation : Import DIDs

Created by ASTPP, last modified on Jul 29, 2015

File must be in the following format(.csv):

DID,Account,Call Type,Destination,Country,Increments,Cost,Setup Fee,Monthly Fee,Included Seconds.

9898589659,3138428084,OTHER,india,india,30,0.5,0.6,1,1

789456123,8800942306,LOCAL,SA,south africa,,30,0.5,0.6,60,1,1

Select provider, csv file of DID and click on Import button, It will display below sample output. You can confirm format if its system recognize your imported file correctly. If all looks good then click on process and system will import all your DIDs.
