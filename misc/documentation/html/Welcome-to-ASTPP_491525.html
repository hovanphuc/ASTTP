ASTPP Documentation : Welcome to ASTPP

Created by Samir Doshi [Administrator], last modified by ASTPP on Jul 30, 2015

ASTPP is an Open Source VoIP Billing Solution for Freeswitch. It supports pre-paid and post-paid billing with call rating and credit control. It also provides many other features such as calling cards, least cost routing (LCR), did management, resellers management etc. http://www.astpp.org
