ASTPP Documentation : Technical Questions

Created by ASTPP, last modified on Jul 23, 2015

1. What databases are supported in ASTPP ?
    It supports only MySQL (PostgreSQL support will be added in future release).
   
2. What payment gateways are supported in ASTPP ?
    Paypal only for now. In future we have plan to add authorize.net payment gateway.

3. Does ASTPP support Multi-language ?
    No
 
4. Does ASTPP 2.0 support callshop ?
    No. It will be included in future releases. To get more information you can contact us at sales@inextrix.com.

5. How can I update my source from to keep it update to date?
    ASTPP provides update.sh script with its source just run that script to update your source.

6. Is calling card module removed in version 2.0?
     No, Calling Card module is integrated with customers. So now account number itself is a Calling Card number.

7. Do we have Oscommerce support in version 2.0?
    Nope, Oscommerce support has been removed in version 2.0. However you may refer older version code and try to implement.
    When I go to login page, rather WEBUI I get long list of php contents.
    Enable short_open_tag in php.ini and then restart apache.

8. When I go to login page, rather WEBUI I get long list of php contents.
    Enable short_open_tag in php.ini and then restart apache.

9. How do we enable call logs ?
    Login into admin portal and go to Settings and Search for name = debug and change its value to 1. Now you will be able to see freeswitch call logs in /var/log/astpp/astpp.log file.
    If file is not created then create it and give 777 permission.

10.How can I access cc_access_numbers variable to configure calling card access numbers?
    You can access system variables from Settings submenu.
    Mouse hover on Welcome Administrator on top right side and that will display drop down menu. Under that you will get Settings submenu.

11.How to change system currency ?
    Change base_currency from Settings which is for database currency reference to update exchange rates. Along with this please also change currency of admin  from Accounts -> Admins -> Edit admin account.

12.Why I am not able to register extensions after installation?
     Make sure you have configured your IP in Sip Profile and your registration request is reaching to server.

13.How to verify if ASTPP and FreeSWITCH communicating properly?
     Default sip profile must be loaded in FreeSWITCH

14.While login I am getting “Unable to connect to your database server using the provided settings” error.
     Please check your database credentials. ASTPP uses /var/lib/astpp/astpp-config.conf file for database connection.

15.How can I do IP Authentication for my customers?
     Configure your customer IP under Customers -> IP Settings

16.Unable to upload large data using csv file in rate module.
     Please increase memory limit from php.ini and restart Apache service.

17.Why I am getting Accountcode is not authenticated error?
     You are using domain name in gateway proxy or in ip settings modules. Please use ip only. Once you change with it, restart freeswitch and test.

18.Why I am getting “Invalid Gateway” error?
     You are using domain name somewhere either in IP Settings or in Gateway proxy. Please change it to IP and restart fs.

19.Getting “500 posting to web server error”
     That means call scripts are not configured correctly. Please make sure to have call scripts in cgi-bin directory. For info please check Apache logs.

20.Httpd not starting after installation and getting “Starting httpd: (13)Permission denied: make_sock: could not bind to address [::]:8081” error.
     Disable selinux and iptables then restart Apache.

21.Does CDRs report will show all data ?
     No, CDRs report will only record of current day. If you want to see record of previous days record then you need to search.
