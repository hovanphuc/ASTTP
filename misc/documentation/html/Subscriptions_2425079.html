ASTPP Documentation : Subscriptions

Created by ASTPP, last modified on Jul 28, 2015

Subscription is kinda package that will be applicable to customer upon invoice generation OR assignment. 

Example : If admin would like to charge customers $100 for Equipment usage on monthly/Daily basis then they can create such type of subscription packages from this module and assign it to entire rate group to individual customers.
